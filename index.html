My Budgeting App

Personal-use app to track finances using the Plaid API.

About

This site exists to satisfy OAuth requirements for certain financial institutions accessed via Plaid. The application itself runs locally on the author's machine and does not provide a hosted user service.

Data & Privacy

OAuth Return URL

If you are testing an OAuth institution via Plaid Link, Plaid will redirect to /plaid/oauth-return.html, which forwards you back to your local application.
